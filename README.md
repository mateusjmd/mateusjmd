# 👨🏽‍💻 Mateus J. Mendes

**`Cientista de Dados`**

Sou graduando no Bacharelado em Ciência e Tecnologia pela [**Ilum – Escola de Ciência**](https://ilum.cnpem.br/), com interesses centrados em Ciência de Dados, Aprendizado de Máquina e Inteligência Artificial. Atualmente sou estagiário no Grupo de Ciência de Dados do [**Laboratório Nacional de Luz Síncrotron**](https://lnls.cnpem.br/), avaliando arquiteturas de Redes Neurais para segmentação de dados volumétricos de tomografia por radiação síncrotron.
